The Nebula

Ultra-high-end glassmorphism Coming Soon template with 12 hand-crafted colour themes, anti-gravity floating field, mouse parallax, live countdown, and AJAX subscription form — all built on Vanilla HTML5, CSS3, and JavaScript. Zero dependencies.

Quick Start

1. Choose your theme

Open any index-0N.html file in your browser. The Gear Switcher (bottom-right corner) lets you navigate between all 12 colour versions.

2. Deploy your chosen theme

Rename your chosen theme file to index.html and upload the entire nebula_html_template/ folder to your server root. Do not change the folder structure — all asset paths are relative.

3. Customise (see sections below)

  • Set your launch date in the data-target-date attribute
  • Add your social profile URLs
  • Wire up your email API endpoint
  • Replace the brand name and logo

Changing a colour palette

Find the relevant body.theme-N { } block in style.css and adjust any of these variables:

Variable Purpose
--clr-a to --clr-e Background nebula gradient stops (dark → light)
--clr-accent Primary accent — links, borders, icons
--clr-text Primary text colour
--clr-text-muted Secondary / caption text
--clr-glass-bg Glass card background (rgba)
--clr-glass-border Glass card border (rgba)
--clr-glass-glow Outer glow / box-shadow colour
--clr-btn-bg / --clr-btn-end Subscribe button gradient

Countdown Date

Find the .countdown element in your HTML and update the data-target-date attribute. Use ISO 8601 UTC format:

<!-- Change the date below to your actual launch date -->
<div class="countdown"
     role="timer"
     data-target-date="2027-01-01T00:00:00Z">
  ...
</div>

The countdown module (assets/js/countdown.js) reads this attribute automatically. When the date is reached, the display resets to 00:00:00:00 and the aria-label updates to "We have launched!"

Social Links

Locate the <nav class="social"> section in your HTML. Replace each href="#" with your real profile URL:

<!-- Example: replace # with your actual URLs -->
<a class="social__link" href="https://twitter.com/yourhandle" ...>
<a class="social__link" href="https://instagram.com/yourhandle" ...>
<a class="social__link" href="https://linkedin.com/in/yourname" ...>
<a class="social__link" href="https://github.com/yourname" ...>
<a class="social__link" href="https://youtube.com/@yourchannel" ...>

To remove a social icon, delete its <a> tag entirely. To add more, copy an existing one and swap the SVG path.

Subscribe Form Integration

The form uses a mock API by default. To connect your real email service, open assets/js/main.js and find the sendToApi() function inside the SubscribeForm module:

// ── REPLACE THIS FUNCTION BODY with your real endpoint ──
const sendToApi = async (email) => {
  // Example: Mailchimp, ConvertKit, custom backend, etc.
  const response = await fetch('https://your-endpoint.com/subscribe', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ email })
  });
  const data = await response.json();
  return { ok: response.ok, message: data.message || 'You are on the list!' };
};

The function must return a Promise that resolves to { ok: boolean, message: string }. The UI handles all loading/success/error states automatically.

Brand & Logo

Changing the site name

Search for The Nebula in your HTML file and replace with your brand name. It appears in three places: the <title> tag, the .brand__name span, and the footer.

Replacing the SVG logo mark

The star icon inside .brand__icon is an inline SVG. Replace it with your own SVG, or use an <img> tag. The icon uses --clr-accent and --clr-e for its gradient, so it will automatically adapt to the active theme.

CSS Architecture

Styles are split into three focused files for clean separation of concerns:

File Contents
style.css CSS reset, design tokens, 12 theme palettes, all component styles (nebula, card, form, social, gear switcher)
animations.css All @keyframes — nebula drift, orb float, geometry shapes, countdown flip, form shake, gear spin
responsive.css All @media queries — mobile (≤480px), tablet (≤768px), 4K (≥2560px), reduced-motion

BEM naming convention is used throughout: .block__element--modifier.

JavaScript Modules

All JavaScript uses Vanilla ES6+ wrapped in IIFEs — no frameworks, no build step.

Module File Purpose
Countdown countdown.js Reads data-target-date, updates digits every second with flip animation
OrbInit main.js Assigns random duration/delay CSS vars to each orb and geometry element
ParallaxField main.js Mouse-driven depth parallax using requestAnimationFrame + lerping. Skipped on touch-only devices.
SubscribeForm main.js Client-side validation, loading state, AJAX API call, success/error feedback
GearSwitcher main.js Opens/closes theme panel, highlights active page link
CopyrightYear main.js Writes current year to #copyrightYear

Performance Notes

  • All animations use only transform and opacity — GPU composited, no layout/paint triggers
  • will-change: transform is applied only to actively animated elements
  • Mouse parallax uses requestAnimationFrame with linear interpolation (lerp) for silk-smooth 60fps
  • Parallax is automatically disabled on touch-only devices (no mouse) and respects prefers-reduced-motion
  • Google Fonts loaded with preconnect hints for faster DNS resolution
  • No JavaScript frameworks, no heavy libraries — total JS < 18KB unminified
